# **App Name**: Secure Talk

## Core Features:

- Real-time Messaging: Send and receive text messages in real-time using Firebase Firestore.
- Contact List: Display a list of dummy contacts. Tapping a contact opens a chat screen.
- Chat Screen UI: Display messages in a chat bubble format with timestamps and distinct styling for sender and receiver messages.
- Navigation Skeleton: Full navigation structure with tabs for Chats, Calls, Nearby,.
- "Coming Soon" Modals: Display a modal indicating "Coming Soon" for unimplemented features like Calls and Nearby.
- Settings Page: Placeholder toggles and buttons in a dummy settings screen.
- README Generation: Generate a README.md file that explains setup steps clearly, including Firebase configuration and build instructions, providing a helpful tool for users.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a secure and trustworthy feel.
- Background color: Light gray (#F0F2F5), a desaturated hue of indigo for a subtle, clean backdrop.
- Accent color: Muted violet (#9575CD) for secondary actions and highlights.
- Body and headline font: 'PT Sans' for a modern and readable appearance.
- Use simple, minimalist icons for navigation and actions.
- Mobile-first, responsive design with a clean and intuitive layout.
- Subtle transitions and animations for a smooth user experience.
